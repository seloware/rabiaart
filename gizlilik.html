Gizlilik Politikası

1. Kişisel Verilerinizin Korunması

RabiaDoğruözArt olarak kişisel verilerinizin gizliliğini önemsiyoruz. Bu gizlilik politikası, web sitemizi kullanırken toplanan bilgilerin nasıl kullanıldığını açıklamaktadır.

2. Toplanan Bilgiler

Web sitemizi ziyaret ettiğinizde aşağıdaki bilgiler toplanabilir:

  • İletişim formunu doldururken verdiğiniz bilgiler
  • IP adresi ve tarayıcı bilgileri
  • Çerezler aracılığıyla toplanan bilgiler

3. Bilgilerin Kullanımı

Toplanan bilgiler şu amaçlarla kullanılmaktadır:

  • Size daha iyi hizmet sunabilmek
  • İletişim taleplerinizi yanıtlamak
  • Web sitemizi geliştirmek ve optimize etmek

4. Bilgi Güvenliği

Kişisel verilerinizin güvenliği için gerekli teknik ve idari önlemleri alıyoruz. Verileriniz üçüncü taraflarla paylaşılmamaktadır.

5. Çerezler

Web sitemiz çerezleri kullanmaktadır. Çerezler, web sitemizi nasıl kullandığınızı anlamamıza ve hizmetlerimizi iyileştirmemize yardımcı olur.
